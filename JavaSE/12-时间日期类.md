# 12 - 时间日期类

## 1. 概述

Java 中的时间日期 API 经历了三个阶段：

| 阶段 | 核心类 | 说明 |
|------|--------|------|
| JDK 1.0 | `Date` | 基础但设计差，大部分方法已废弃 |
| JDK 1.1 | `Calendar` | 替代 Date，但依然难用 |
| JDK 8+ | `LocalDate / LocalTime / LocalDateTime` | 新 API，不可变、线程安全、推荐使用 |

---

## 2. Date 类（java.util.Date）

### 2.1 创建对象

```java
// 当前时间
Date now = new Date();
System.out.println(now);  // Sun Jul 19 15:22:50 CST 2026

// 指定毫秒数（从 1970-01-01 00:00:00 UTC 开始）
Date date = new Date(1000L * 60 * 60 * 24);  // 1970-01-02
```

### 2.2 常用方法

```java
Date d1 = new Date();
Date d2 = new Date(System.currentTimeMillis());

// 获取毫秒值
long time = d1.getTime();  // 推荐

// 比较
d1.before(d2);   // 是否在 d2 之前
d1.after(d2);    // 是否在 d2 之后
d1.compareTo(d2); // 负数/0/正数
```

> ⚠️ `getYear()`、`getMonth()` 等方法已废弃，不要再用了

---

## 3. SimpleDateFormat（日期格式化）

`java.text.SimpleDateFormat` 用于 Date 和字符串之间的互转。

### 3.1 常用格式符号

| 符号 | 含义 | 示例 |
|------|------|------|
| `y` | 年 | 2026 |
| `M` | 月 | 07 |
| `d` | 日 | 19 |
| `H` | 时（0-23） | 15 |
| `m` | 分 | 22 |
| `s` | 秒 | 50 |
| `S` | 毫秒 | 123 |
| `E` | 星期几 | 周日 / Sunday |
| `a` | 上午/下午 | 下午 |

### 3.2 使用示例

```java
// Date → String
Date now = new Date();
SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
String str = sdf.format(now);  // "2026-07-19 15:22:50"

// String → Date
String dateStr = "2026-07-19 15:22:50";
Date date = sdf.parse(dateStr);  // 需要处理 ParseException
```

### 3.3 注意事项

- **线程不安全**：`SimpleDateFormat` 不是线程安全的，不要在并发环境下共享使用
- **异常处理**：`parse()` 方法抛出 `ParseException`，需要 try-catch
- **严格模式**：默认宽松解析（2026-13-01 会变成 2027-01-01），可设置：

```java
sdf.setLenient(false);  // 严格模式，非法日期抛异常
```

## 4. Calendar 类（java.util.Calendar）

### 4.1 基本使用

```java
// 获取 Calendar 实例
Calendar cal = Calendar.getInstance();

// 获取年月日
int year  = cal.get(Calendar.YEAR);
int month = cal.get(Calendar.MONTH) + 1;     // ⚠️ 0-11，要 +1
int day   = cal.get(Calendar.DAY_OF_MONTH);
int hour  = cal.get(Calendar.HOUR_OF_DAY);    // 24小时制
int min   = cal.get(Calendar.MINUTE);
int sec   = cal.get(Calendar.SECOND);
int week  = cal.get(Calendar.DAY_OF_WEEK);    // 1=周日, 2=周一...

// 设置时间
cal.set(Calendar.YEAR, 2027);
cal.set(2026, Calendar.DECEMBER, 25);  // 月用常量
```

### 4.2 日期加减

```java
Calendar cal = Calendar.getInstance();

// 加 7 天
cal.add(Calendar.DAY_OF_MONTH, 7);

// 减 3 个月
cal.add(Calendar.MONTH, -3);

// set() 会直接覆盖，add() 会进位/借位
cal.set(Calendar.DAY_OF_MONTH, 35);  // 自动变成下个月的某一天
```

### 4.3 Calendar vs Date 转换

```java
// Calendar → Date
Date date = cal.getTime();

// Date → Calendar
cal.setTime(new Date());
```

> ⚠️ Calendar 也是**可变对象**且 API 设计别扭，不推荐在新代码中使用。

## 5. JDK 8 新时间 API（java.time）

从 JDK 8 开始引入 `java.time` 包，解决了旧 API 的所有痛点：
- **不可变**（线程安全）
- **设计清晰**（日期、时间、日期时间分三类）
- **操作丰富**（加减、比较、格式化）

### 5.1 LocalDate（本地日期）

```java
// 获取当前日期
LocalDate today = LocalDate.now();             // 2026-07-19
LocalDate date  = LocalDate.of(2026, 7, 19);   // 指定日期

// 获取年/月/日
int year  = today.getYear();                   // 2026
int month = today.getMonthValue();             // 7
int day   = today.getDayOfMonth();             // 19
DayOfWeek week = today.getDayOfWeek();         // SUNDAY
int dayOfYear = today.getDayOfYear();          // 200

// 判断
today.isBefore(date);
today.isLeapYear();  // 闰年判断

// 加减
today.plusDays(1);             // 明天
today.minusMonths(3);
today.plusWeeks(2);
today.withDayOfMonth(1);       // 当月第一天
today.withMonth(12);           // 改为12月（返回新对象）
```

### 5.2 LocalTime（本地时间）

```java
LocalTime now = LocalTime.now();              // 15:22:50.123
LocalTime time = LocalTime.of(15, 22, 50);    // 15:22:50

int hour = now.getHour();
int min  = now.getMinute();
int sec  = now.getSecond();

now.plusHours(2);
now.minusMinutes(30);
```

### 5.3 LocalDateTime（日期+时间）

```java
LocalDateTime now = LocalDateTime.now();
LocalDateTime dt  = LocalDateTime.of(2026, 7, 19, 15, 22, 50);

// LocalDateTime ↔ LocalDate / LocalTime
LocalDate date = now.toLocalDate();
LocalTime time = now.toLocalTime();

// 组合
LocalDateTime from = LocalDateTime.of(date, time);

// 加减
dt.plusDays(7).minusHours(3);  // 链式调用
```

### 5.4 DateTimeFormatter（格式化，线程安全）

```java
// 方式一：使用预定义常量
DateTimeFormatter fmt1 = DateTimeFormatter.ISO_DATE;      // 2026-07-19
DateTimeFormatter fmt2 = DateTimeFormatter.ISO_DATE_TIME; // 2026-07-19T15:22:50

// 方式二：自定义格式（推荐）
DateTimeFormatter fmt = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss");

// LocalDateTime → String
LocalDateTime now = LocalDateTime.now();
String str = now.format(fmt);  // "2026-07-19 15:22:50"

// String → LocalDateTime
String dateStr = "2026-07-19 15:22:50";
LocalDateTime dt = LocalDateTime.parse(dateStr, fmt);
```

### 5.5 Instant（时间戳）

`Instant` 表示时间线上的一个瞬时点，精度到纳秒，常用于做时间戳和跨时区计算。

```java
// 当前时间戳（UTC）
Instant now = Instant.now();                       // 2026-07-19T07:22:50Z

// Instant ↔ Date（互转）
Instant ins = new Date().toInstant();
Date date = Date.from(Instant.now());

// Instant ↔ 毫秒
long millis = Instant.now().toEpochMilli();
Instant fromMillis = Instant.ofEpochMilli(millis);

// 秒级时间戳
long seconds = Instant.now().getEpochSecond();
```

### 5.6 Duration 与 Period（时间差）

```java
// Duration —— 计算时分秒纳秒级别的差值
LocalDateTime start = LocalDateTime.of(2026, 7, 1, 8, 0);
LocalDateTime end   = LocalDateTime.of(2026, 7, 19, 15, 22);

Duration duration = Duration.between(start, end);
long days    = duration.toDays();        // 18
long hours   = duration.toHours();       // 439
long minutes = duration.toMinutes();     // 26362

// Period —— 计算年月日级别的差值
LocalDate d1 = LocalDate.of(2025, 7, 19);
LocalDate d2 = LocalDate.of(2026, 7, 19);

Period period = Period.between(d1, d2);
period.getYears();   // 1
period.getMonths();  // 0
period.getDays();    // 0
```

### 5.7 ZoneId / ZonedDateTime（时区处理）

```java
// 所有可用时区
Set<String> zones = ZoneId.getAvailableZoneIds();

// 指定时区
ZoneId shanghai = ZoneId.of("Asia/Shanghai");
ZoneId utc      = ZoneId.of("UTC");

// 本地时间 → 带时区时间
LocalDateTime local = LocalDateTime.now();
ZonedDateTime zdt = local.atZone(shanghai);  // 2026-07-19T15:22:50+08:00[Asia/Shanghai]

// 不同时区时间互转
ZonedDateTime tokyo = zdt.withZoneSameInstant(ZoneId.of("Asia/Tokyo"));
```

### 5.8 TemporalAdjusters（日期调整器）

```java
LocalDate today = LocalDate.now();

// 下个周一
LocalDate nextMonday = today.with(TemporalAdjusters.next(DayOfWeek.MONDAY));

// 本月最后一天
LocalDate lastDayOfMonth = today.with(TemporalAdjusters.lastDayOfMonth());

// 本月第一个周日
LocalDate firstSunday = today.with(TemporalAdjusters.firstInMonth(DayOfWeek.SUNDAY));
```

## 6. 时间 API 对比汇总

| 操作 | 旧 API（不推荐） | JDK 8 API（推荐） |
|------|-----------------|-------------------|
| 创建当前时间 | `new Date()` | `LocalDateTime.now()` |
| 格式化 | `SimpleDateFormat`（线程不安全） | `DateTimeFormatter`（线程安全） |
| 日期加减 | `Calendar.add()` | `plusDays()` / `minusMonths()` |
| 时间差 | 自己算毫秒 | `Duration` / `Period` |
| 时区 | `TimeZone` | `ZoneId` / `ZonedDateTime` |
| 闰年判断 | 自己写 | `LocalDate.isLeapYear()` |
| 日期调整 | 手算 | `TemporalAdjusters` |

## 7. 系统级时间

```java
// 获取当前时间的毫秒值（常用）
long start = System.currentTimeMillis();
// ... 执行代码 ...
long end = System.currentTimeMillis();
System.out.println("耗时: " + (end - start) + "ms");

// 纳秒级计时（更精确的性能测量，不是墙上时间）
long nano = System.nanoTime();
```

---

## 8. 总结

1. **JDK 8 之前**：`Date` + `SimpleDateFormat` + `Calendar`，难用且线程不安全
2. **JDK 8+**：`LocalDate` / `LocalTime` / `LocalDateTime`，推荐使用
3. **格式化**：`DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss")`
4. **时间差**：`Duration`（时分秒）、`Period`（年月日）
5. **旧项目维护**：会用 `SimpleDateFormat` 和 `Calendar` 就行，新代码一律用 JDK 8 新 API
