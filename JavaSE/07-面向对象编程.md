# 07 - 面向对象编程 (OOP)

## 1. 面向对象概述

### 1.1 面向对象 vs 面向过程

```java
// 面向过程：关注"怎么做"——按步骤执行
// 洗衣服：1. 放水  2. 放衣服  3. 倒洗衣液  4. 搓  5. 漂洗  6. 拧干

// 面向对象：关注"谁来做"——找对象干活
// 洗衣服：洗衣机.洗衣服(衣服, 洗衣液)
```

| 对比 | 面向过程（POP） | 面向对象（OOP） |
|------|----------------|----------------|
| 关注点 | 步骤/流程 | 对象/类 |
| 核心 | 函数/方法 | 类/对象 |
| 特点 | 自顶向下逐步细化 | 封装继承多态 |
| 维护性 | 需求变化需要改多处 | 模块化，改局部不影响整体 |
| 适用 | 简单、流程固定的场景 | 复杂、大型业务系统 |

### 1.2 类与对象

- **类（class）**：是对一类事物的**描述**，是抽象的，相当于**图纸**
- **对象（object）**：是类的**具体实例**，是具体的，相当于**按图纸造出来的实物**

```java
// 类：描述"手机"这一类事物
public class Phone {
    // 属性（成员变量）
    String brand;
    double price;
    String color;

    // 行为（成员方法）
    public void call(String name) {
        System.out.println("给" + name + "打电话");
    }

    public void playGame() {
        System.out.println("玩游戏");
    }
}
```

## 2. 对象的使用

### 2.1 创建对象

```java
// 格式：类名 对象名 = new 类名();
Phone p = new Phone();
```

### 2.2 使用对象

```java
// 访问属性：对象名.属性名
p.brand = "华为";
p.price = 5999.0;
p.color = "白色";

// 调用方法：对象名.方法名(参数)
p.call("张三");     // 输出：给张三打电话
p.playGame();       // 输出：玩游戏
```

### 2.3 同一个类创建多个对象

```java
Phone p1 = new Phone();
Phone p2 = new Phone();

p1.brand = "华为";
p2.brand = "苹果";

System.out.println(p1.brand);  // 华为
System.out.println(p2.brand);  // 苹果
// 每个对象独立拥有一份属性副本
```

### 2.4 对象内存图解

```

│ main    │              │ Phone 对象 #1 │
│ p1 ────┼──→ [0x100]  │ brand = "华为"│
│ p2 ────┼──→ [0x200]  │ price = 5999  │
│         │              │ color = "白色"│
│         │              └──────────────┘
│         │              ┌──────────────┐
│         │              │ Phone 对象 #2 │

     ↓                         ↑
  栈中的变量 p1/p2       堆中的实际对象
  存的是对象的地址       （成员变量存在这里）
```

## 3. 成员变量与局部变量

### 3.1 定义位置

```java
public class Student {
    String name;       // 成员变量：类中，方法外

    public void study() {
        int hour = 8;  // 局部变量：方法内
        System.out.println(name + "学习了" + hour + "小时");
    }
}
```

### 3.2 对比

| 区别 | 成员变量 | 局部变量 |
|------|---------|---------|
| **定义位置** | 类中，方法外 | 方法内/方法参数/代码块内 |
| **默认值** | 有默认值（`null`, `0`, `false`） | 无默认值，必须先赋值 |
| **生命周期** | 随对象存在，对象回收才消失 | 方法执行结束即消失 |
| **存储位置** | 堆（对象中） | 栈（方法栈帧中） |

## 4. 封装（Encapsulation）

### 4.1 为什么要封装？

```java
// 不封装的问题：属性可以直接被随意篡改
public class Student {
    String name;
    int age;
}

Student s = new Student();
s.age = -100;  // 年龄为负数，不合理！但无法阻止
```

### 4.2 封装的步骤

**① `private` 关键字**：将成员变量私有化，外界无法直接访问
**② `getter/setter` 方法**：提供公共的访问方法，可以在方法内加校验逻辑

```java
public class Student {
    // 1. 私有化成员变量
    private String name;
    private int age;

    // 2. 提供公共的 getter/setter
    public void setAge(int age) {
        // 3. 在 setter 中加入校验逻辑
        if (age >= 0 && age <= 150) {
            this.age = age;
        } else {
            System.out.println("年龄不合法：" + age);
        }
    }

    public int getAge() {
        return age;
    }

    public void setName(String name) {
        this.name = name;
    }

    public String getName() {
        return name;
    }
}
```

### 4.3 封装的优点

| 优点 | 说明 |
|------|------|
| **安全性** | 防止非法数据写入（通过校验） |
| **可维护** | 内部逻辑修改不影响外部使用 |
| **复用性** | 封装好的类可以到处使用 |
| **低耦合** | 使用者只需关注公共方法，无需了解内部实现 |

## 5. `this` 关键字

### 5.1 作用：区分同名变量

```java
public class Student {
    private String name;  // 成员变量

    public void setName(String name) {  // 参数名也叫 name
        // name = name;  // 问题：两个 name 都是局部变量（就近原则）
        this.name = name;  // this.name → 成员变量；name → 局部变量（参数）
    }
}
```

### 5.2 `this` 的本质

> `this` 就是**当前对象的引用**，谁调用了这个方法，`this` 就指向谁。

```java
public class Student {
    private String name;

    public void printThis() {
        System.out.println(this);  // 输出类似：Student@15db9742
    }
}

public static void main(String[] args) {
    Student s1 = new Student();
    Student s2 = new Student();

    s1.printThis();  // this = s1 的地址
    s2.printThis();  // this = s2 的地址（不同！）
}
```

## 6. 构造方法

### 6.1 什么是构造方法？

构造方法是**创建对象时**自动调用的特殊方法。

```java
// 格式：public 类名(参数列表) { 方法体 }
public class Student {
    private String name;
    private int age;

    // 无参构造
    public Student() {
        System.out.println("无参构造被调用了");
    }

    // 带参构造：创建对象时直接赋值
    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }
}
```

### 6.2 构造方法的细节

- **方法名必须与类名相同**
- **没有返回值类型，连 `void` 也不写**
- 定义构造方法时不能加 `return` 语句
- **如果没有定义任何构造方法，编译器会默认生成一个无参构造**
- **如果定义了带参构造，无参构造不会自动生成，需要手动写**

```java
Student s1 = new Student();                     // 调用无参构造
Student s2 = new Student("张三", 20);           // 调用带参构造
```

### 6.3 构造方法的重载

```java
public class Student {
    private String name;
    private int age;

    // 无参构造
    public Student() {}

    // 单参构造
    public Student(String name) {
        this.name = name;
    }

    // 全参构造
    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }
}
```

## 7. 标准 JavaBean

### 7.1 规范

一个标准的 JavaBean 需要满足三个条件：

1. **成员变量使用 `private` 修饰**（封装）
2. **提供 `getter` / `setter` 方法**
3. **提供无参构造**（推荐同时也提供全参构造）

```java
public class User {
    // 1. private 成员变量
    private String username;
    private String password;
    private String email;

    // 2. 无参构造（必须）
    public User() {}

    // 全参构造（推荐）
    public User(String username, String password, String email) {
        this.username = username;
        this.password = password;
        this.email = email;
    }

    // 3. getter / setter
    public String getUsername() {
        return username;
    }

    public void setUsername(String username) {
        this.username = username;
    }

    public String getPassword() {
        return password;
    }

    public void setPassword(String password) {
        this.password = password;
    }

    public String getEmail() {
        return email;
    }

    public void setEmail(String email) {
        this.email = email;
    }
}
```

### 7.2 三种赋值方式对比

| 方式 | 代码 | 特点 |
|------|------|------|
| 直接属性赋值 | `对象.属性 = 值` | 最简单，但不安全 |
| setter 方法 | `对象.setXxx(值)` | 可加校验，最常用 |
| 构造方法 | `new 类(参数...)` | 创建对象时一步到位 |

## 8. 对象数组

### 8.1 定义与使用

```java
// 1. 定义学生数组：长度为 3
Student[] students = new Student[3];

// 2. 创建学生对象
Student s1 = new Student("张三", 20);
```

## 9. 面向对象三大特征简介

Java 面向对象的三大特征：**封装、继承、多态**（后续章节详细讲）。

- **封装**：本节课已学，用 `private` 保护数据
- **继承**：子类继承父类的属性和方法

## 10. API 与 ArrayList 入门

### 10.1 API 是什么

API（Application Programming Interface）：Java 官方提供的**现成的类和方法**，直接拿来用。

### 10.2 ArrayList — 可变数组

#### 为什么需要 ArrayList？

```java
// 普通数组：长度固定，无法改变
int[] arr = new int[3];  // 永远只能存 3 个

// ArrayList：长度自动增长
ArrayList<String> list = new ArrayList<>();
list.add("Hello");
list.add("World");
list.add("Java");
list.add("!");      // 没问题！自动扩容
```

#### 创建 ArrayList

```java
import java.util.ArrayList;

// ArrayList<数据类型> 变量名 = new ArrayList<>();
ArrayList<String> list = new ArrayList<>();       // 只能存 String
ArrayList<Integer> list2 = new ArrayList<>();     // 只能存 Integer（包装类）
```

> `<泛型>` 必须使用**引用类型**，不能用基本类型（int → Integer, double → Double 等）

#### 常用方法

```java
ArrayList<String> list = new ArrayList<>();

// 添加元素
boolean result = list.add("Hello");  // 尾部添加，永远返回 true

// 获取元素
String s = list.get(0);              // 按索引取

// 删除元素
list.remove(0);                      // 按索引删除，返回被删元素
list.remove("Hello");                // 按内容删除，返回 boolean

// 修改元素
String old = list.set(0, "World");  // 替换，返回被替换的旧元素

// 获取长度
int size = list.size();              // 注意：这是 size()，不是数组的 length
```

#### 遍历

```java
ArrayList<String> list = new ArrayList<>();
list.add("Apple");
list.add("Banana");
list.add("Cherry");

// 普通 for
for (int i = 0; i < list.size(); i++) {
    System.out.println(list.get(i));
}

// 增强 for（for-each）
for (String s : list) {
    System.out.println(s);
}
```

## 11. 学生管理系统（案例）

### 11.1 需求

实现一个简单的学生管理系统，支持：

1. 添加学生
2. 删除学生
3. 修改学生信息
4. 查询所有学生
5. 退出系统

### 11.2 学生类

```java
public class Student {
    private String id;      // 学号（唯一标识）
    private String name;
    private int age;
    private String address; // 住址

    public Student() {}
    public Student(String id, String name, int age, String address) {
        this.id = id;
        this.name = name;
        this.age = age;
        this.address = address;
    }

    // getter / setter（省略完整代码）
    public String getId() { return id; }
    public void setId(String id) { this.id = id; }

    public String getName() { return name; }
    public void setName(String name) { this.name = name; }

    public int getAge() { return age; }
    public void setAge(int age) { this.age = age; }

    public String getAddress() { return address; }
    public void setAddress(String address) { this.address = address; }
}
```

### 11.3 管理系统主逻辑

```java
import java.util.ArrayList;
import java.util.Scanner;

public class StudentSystem {
    public static void main(String[] args) {
        ArrayList<Student> list = new ArrayList<>();
        Scanner sc = new Scanner(System.in);

        loop:
        while (true) {
            System.out.println("===== 学生管理系统 =====");
            System.out.println("1. 添加学生");
            System.out.println("2. 删除学生");
            System.out.println("3. 修改学生");
            System.out.println("4. 查询所有学生");
            System.out.println("5. 退出");
            System.out.print("请选择：");

            int choice = sc.nextInt();
            switch (choice) {
                case 1 -> addStudent(list, sc);
                case 2 -> deleteStudent(list, sc);
                case 3 -> updateStudent(list, sc);
                case 4 -> queryAllStudents(list);
                case 5 -> {
                    System.out.println("感谢使用，再见！");
                    break loop;
                }
                default -> System.out.println("输入有误，请重新选择");
            }
        }
    }

    // 查询所有学生
    private static void queryAllStudents(ArrayList<Student> list) {
        if (list.isEmpty()) {
            System.out.println("当前无学生信息，请添加后再查询");
            return;
        }
        System.out.println("学号\t姓名\t年龄\t住址");
        for (Student s : list) {
            System.out.println(s.getId() + "\t"
                    + s.getName() + "\t"
                    + s.getAge() + "\t"
                    + s.getAddress());
        }
    }

    // 添加学生
    private static void addStudent(ArrayList<Student> list, Scanner sc) {
        System.out.print("请输入学号：");
        String id = sc.next();
        // 学号唯一性校验
        if (getIndex(list, id) >= 0) {
            System.out.println("学号已存在，添加失败");
            return;
        }
        System.out.print("请输入姓名：");
        String name = sc.next();
        System.out.print("请输入年龄：");
        int age = sc.nextInt();
        System.out.print("请输入住址：");
        String address = sc.next();

        Student s = new Student(id, name, age, address);
        list.add(s);
        System.out.println("添加成功");
    }

    // 删除学生
    private static void deleteStudent(ArrayList<Student> list, Scanner sc) {
        System.out.print("请输入要删除的学号：");
        String id = sc.next();
        int index = getIndex(list, id);
        if (index < 0) {
            System.out.println("未找到该学生");
            return;
        }
        list.remove(index);
        System.out.println("删除成功");
    }

    // 修改学生
    private static void updateStudent(ArrayList<Student> list, Scanner sc) {
        System.out.print("请输入要修改的学号：");
        String id = sc.next();
        int index = getIndex(list, id);
        if (index < 0) {
            System.out.println("未找到该学生");
            return;
        }
        Student s = list.get(index);
        System.out.print("请输入新姓名（当前：" + s.getName() + "）：");
        s.setName(sc.next());
        System.out.print("请输入新年龄（当前：" + s.getAge() + "）：");
        s.setAge(sc.nextInt());
        System.out.print("请输入新住址（当前：" + s.getAddress() + "）：");
        s.setAddress(sc.next());
        System.out.println("修改成功");
    }

    // 根据学号查找索引
    private static int getIndex(ArrayList<Student> list, String id) {
        for (int i = 0; i < list.size(); i++) {
            if (list.get(i).getId().equals(id)) {
                return i;
            }
        }
        return -1;
    }
}
```

## 12. 总结

| 知识点 | 核心要点 |
|--------|----------|
| 类与对象 | 类是图纸，对象是实物；一个类可创建多个对象 |
| 成员变量 | 堆中存储，有默认值，随对象存在 |
| 局部变量 | 栈中存储，无默认值，方法结束即消失 |
| 封装 | `private` + `getter/setter`，保护数据安全 |
| `this` | 当前对象的引用，用于区分同名变量 |
| 构造方法 | 类名同名，无返回值，重载灵活 |
| 标准 JavaBean | `private` 变量 + 无参构造 + getter/setter |
| 对象数组 | 数组中存放对象的地址 |
| ArrayList | 可变数组，支持 `<泛型>`，常用 `add/get/remove/set/size` |
| 学生管理系统 | 综合运用类 + ArrayList + Scanner 实现增删改查 |
